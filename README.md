**makro** is a custom macropad with hotswap keys and a built-in USB hub with remap port.

## üîß Features
### **Hot-Swappable Keys**
Customize the sound and feel of your macropad by swapping out switches without soldering.

### **Programmable USB Hub**
Rebind any HID (Human Interface Device) plugged into the remap port, enabling advanced customization of input devices like keyboards, mice, or game controllers.

### **LED Lighting**
Features optional per-switch gradients and diffused backlighting for a visually appealing setup that enhances your workspace aesthetics.
